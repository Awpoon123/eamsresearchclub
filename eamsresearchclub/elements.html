Courses

Notebook 1

Our first notebook introduces users to basic data exploration concepts using the life expectancy dataset. It provides clear explanations of R commands and includes visualizations to help illustrate the data effectively.

Datsets

https://www.kaggle.com/datasets/kumarajarshi/life-expectancy-who

Note

The Dataset used in the notebooks is a cleaned version of the data on Kaggle.

Notebook 2

Our second notebook teaches user to use R to create scatterplots, fit simple linear regression models, and compare the strength of models. Users will be able to see which features affect the target variable in a dataset and how.

Notebook 3

Our third notebook explores the logistics behind linear regression and building effective machine learning models. Users are tasked with constructing these models to make accurate predictions related to the dataset.

Notebook 4

Our third notebook shows users how to use R to add polynomial terms to your multiple regression models which is known as polynomial regression. Then users will use the principles of machine learning to tune models to predict target values based on unseen data.
